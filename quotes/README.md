## Quote Format

Each quote includes:
- **Full quotation** in blockquote format
- **Page number** for citation
- **Context** - where in source this appears
- **Relevance** - why this matters for dissertation
- **Dimensions** - which of 5 dimensions it addresses
- **Chapter Placement** - suggested location(s)
- **Integration Notes** - connections to other sources/chapters

---

## Usage Guidelines

### For Writing:
1. Search by keyword or theme
2. Copy quote with attribution
3. Reference page number for citation
4. Check integration notes for context

### For Citations:
- Use APA format: (Author, Year, p. XX)
- Full source in `bibliography/references.bib`
- Verify quote accuracy before using
