# Reading Notes by Type

This section organizes notes by category rather than by source or chapter.

## Categories

### [Ideas and Insights](ideas.md)
- Research ideas
- Theoretical connections
- Novel hypotheses
- Integration opportunities

### [Advisor Comments](advisor_comments.md)
- Meeting notes with advisor
- Feedback and revisions requested
- Questions to address
- Methodological guidance

### [Methodological Notes](methodology.md)
- Research design decisions
- Data collection strategies
- Analysis approaches
- Philosophical frameworks

## How to Use

- **Ideas**: Capture spontaneous insights, connections between sources, novel theoretical contributions
- **Advisor Comments**: Track all advisor feedback in chronological order for easy reference
- **Methodology**: Document methodological decisions and rationale
