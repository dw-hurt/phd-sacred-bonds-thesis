# Integration Notes

This folder contains AI-processed summaries and syntheses from research journal entries. These notes bridge raw ideation and formal chapter writing.

## Structure

### By Chapter
- **`by_chapter/chapter_XX/`** - Summaries organized by chapter
- Each file represents processed insights from journal entries

### By Theme
- **`by_theme/`** - Cross-chapter thematic integrations
- Connects ideas that span multiple chapters

## Workflow

1. **Write** freely in `research_journal/`
2. **Request** AI extraction (specify chapter/theme)
3. **Review** AI-generated summaries here
4. **Reference** these notes when writing formal chapters
